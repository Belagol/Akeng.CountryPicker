# Contributing to Akeng.CountryPicker

First of all, thank you for considering contributing to **Akeng.CountryPicker**!

We welcome contributions of all kinds, including:

- 🐞 Bug reports
- 💡 Feature requests
- 📖 Documentation improvements
- ✨ New features
- 🛠 Performance improvements
- 🧪 Unit tests

---

## Getting Started

1. Fork the repository.

2. Clone your fork.

```bash
git clone https://github.com/Belagol/Akeng.CountryPicker.git
```

3. Create a new branch.

```bash
git checkout -b feature/my-feature
```

4. Implement your changes.

5. Commit your work.

```bash
git commit -m "Add amazing feature"
```

6. Push your branch.

```bash
git push origin feature/my-feature
```

7. Open a Pull Request.

---

## Coding Guidelines

Please follow these recommendations:

- Use meaningful names.
- Keep methods short and focused.
- Follow C# naming conventions.
- Use nullable reference types.
- Add XML documentation for public APIs.
- Keep the package lightweight and dependency-free whenever possible.

---

## Pull Requests

Please ensure that:

- The project builds successfully.
- Existing functionality is not broken.
- New public APIs are documented.
- The README is updated when necessary.

---

## Reporting Issues

When opening an issue, please include:

- Platform (Android, iOS, Windows, MacCatalyst)
- .NET MAUI version
- Package version
- Steps to reproduce
- Expected behavior
- Actual behavior
