# Code of Conduct

## Our Pledge

We are committed to making participation in this project a welcoming and respectful experience for everyone.

Regardless of:

- Age
- Experience
- Nationality
- Gender
- Disability
- Religion

everyone is welcome.

---

## Expected Behavior

Please:

- Be respectful.
- Be constructive.
- Help newcomers.
- Accept feedback professionally.
- Focus on improving the project.

---

## Unacceptable Behavior

Examples include:

- Harassment
- Personal attacks
- Hate speech
- Discrimination
- Spam
- Offensive language

These behaviors will not be tolerated.

---

## Enforcement

Project maintainers reserve the right to remove comments, issues, pull requests, or contributors that violate this Code of Conduct.

---

Thank you for helping build a positive open-source community.
